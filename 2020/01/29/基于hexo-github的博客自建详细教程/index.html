前期准备：

  1. 安装Node.js
  2. 安装Git
  3. 安装一个文本编辑器（根据自己的需求安装，自用Sublime Text3）

hexo安装详细步骤：（windows按安装步骤）

npm install hexo-cli -g
本地创建一个文件夹，此处以blog为例
进入到blog目录下
hexo init blog
npm install
hexo s
打开浏览器127.0.0.1：4000，出现hexo页面，如图（页面不是我这个主题，能成功打开即可）

hexo使用详细说明：

hexo n "name"
用文本编辑器编辑页面（附Markdown 语法说明） 
hexo g
hexo s
打开127.0.0.1:4000
hexo d(上传到github上，后续讲解)

Markdown 语法说明

hexo上载到github详细教程：

在blog下安装部署插件：cpm install --save hexo-deployer
修改_config.yml配置文件，如下
        # Deployment
        ## Docs: https://hexo.io/docs/deployment.html
        deploy:
            type: git
            repo: https://github.com/gjxvictory/gjxvictory.github.io.git(git仓库连接)
            babranch: master
hexo d

主题安装

在hexo官网下载主题，解压到\blog\themes中
修改_config.yml配置文件，如下
        theme: fluid

在hexo中插入图片：

npm install hexo-asset-image --save
修改_config.yml配置文件，如下
        post_asset_folder: true
将图片放入\blog\source\_posts的同名文件夹下
Markdown语法：![你想输入的替代文字](图片名.jpg)
